Every time there is an update to the models, an event is fired.

The payload will always look like this, where the body: Model is described with the following interface

interface Message {
    id: string
    body: Model
    index: string
}
assembly done

.add .remove .update

interface Assembly{
        assembly_id: number,
        from: string //('Y-m-d'),
        to: string | null //('Y-m-d'),
    }
